Política de Privacidade

Sua privacidade é importante para nós. Esta política explica como coletamos, usamos e protegemos seus dados no Ali Software Jurídico.

Coleta de Dados

Coletamos apenas os dados necessários para funcionamento do sistema, como nome, email, OAB, e dados de clientes/processos cadastrados por você.

Uso dos Dados

Os dados são utilizados exclusivamente para prestação dos serviços contratados, nunca sendo compartilhados com terceiros sem consentimento.

Segurança

Adotamos medidas técnicas e organizacionais para proteger seus dados contra acesso não autorizado, perda ou divulgação indevida.

Direitos do Usuário

Você pode solicitar a exclusão ou portabilidade dos seus dados a qualquer momento, entrando em contato pelo email isabela@alisoftware.com.br.
